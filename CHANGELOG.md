# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-05-24
### Added
- Created professional repository standard files: `.gitignore`, `README.md`, `LICENSE`, `CONTRIBUTING.md`, `SECURITY.md`, and `CHANGELOG.md`.
- Structured standard `docs/` folder in the root directory containing architectural and promotion reports.
- Added product promo screenshots to the portfolio.

### Fixed
- Standardized git upload boundaries, safely excluding local environment settings and temporary OS junk.

## [1.0.0] - 2023-11-20
### Added
- Completed initial release of **Metal Wheels** static HTML/CSS web application.
- Added car brand landing pages for BMW, Rolls Royce, Porsche, and Lamborghini.
- Integrated autoplay and loop MP4 teaser videos for car cards.
- Integrated a customized contact page and interactive review submit form.
- Embedded interactive Google Maps location widget.
- Created custom branding and logos.
