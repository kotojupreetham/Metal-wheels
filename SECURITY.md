# Security Policy

## Supported Versions

This is a static portfolio site created in 2023 for educational purposes. We only provide security support for the latest pushed version in the repository.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you identify any security issue (e.g., exposed keys, problematic iframe integrations, cross-site scripting in custom scripts, or dangerous external resources), please report it to us directly.

**Do not open a public issue.** Instead, please follow these steps:

1. Send an email to the project maintainer (see contact details in [README.md](README.md)).
2. Describe the vulnerability, listing the affected file(s) and a proof of concept or steps to reproduce.
3. We will respond within 48–72 hours to coordinate a fix.

Thank you for helping keep this educational portfolio safe!
