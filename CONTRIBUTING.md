# Contributing to Metal Wheels

Thank you for visiting this repository! 

## Historical Preservation Policy
This repository hosts a portfolio project originally created in **2023** during the author's **B.Tech 1st year**. 
To preserve the authenticity, history, and raw educational value of the author's first self-built, AI-free coding project, **direct code contributions, feature requests, or pull requests to modify the codebase are closed.**

## How to Engage
Even though direct code changes are not accepted, your feedback and engagement are highly appreciated! You can:
1. **Submit Issues:** If you find critical bugs, browser compatibility issues, or security concerns, feel free to open a GitHub Issue.
2. **Discussions:** If you have questions about the design, styling choices, or implementation details of this 2023 project, feel free to initiate a GitHub Discussion.
3. **Portfolio Reviews:** If you are a recruiter, senior engineer, or portfolio reviewer, your feedback on code quality, architecture, and UI/UX design is extremely valuable. Feel free to reach out via the contacts listed in the [README.md](README.md).

Thank you for respecting the preservation of this early-career milestone project!
